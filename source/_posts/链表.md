# O(1)时间删除节点
  - 解法
    - 将要删除节点的下一个节点的值复制到要删除的节点
    - 将要删除节点的指针指向其下一个节点的下一个节点

  - 理解：重点在于，不是物理上去删除这个节点，而是从逻辑上删除了这个节点

  - 代码

    ```c++
    Node* delete(Node* ptr, Node* head){
        Node* next = ptr->next;
        ptr->val = next->val;
        ptr->next = next->next;
        delete next;
        return head;
    }
    ```
